## READ ME
The example input file is in the XGB_Input folder. The '8_PatientLevel_char_WithPossibleMonthsHasNoCodes.xlsx' is generated by step 8 and the 'All_11E_CCSandVAL2nd.pkl' is generated by step 11 using the data preprocessing code.
The XGBoost models you need are saved in the Saved_XGBoost folder. Please do not change anything in this folder.
The codes you need are saved in the Code folder. You could use the following command for testing.

Example: I want to use SBCE_Excluded_DeathPts definition for y SBCE patients, use the VALand drug code, Full model and a cutoff of 0.6

python3 Test_onlyPrediction.py -indir ../XGB_Input -outdir ../results -model ../Saved_XGBoost -fs CCSandVAL2nd -sc SBCE_Excluded_DeathPts -sm Full_Model -inputname All_11E_CCSandVAL2nd.pkl -chrname 8_PatientLevel_char_WithPossibleMonthsHasNoCodes.xlsx -cutoff 0.6


python3 Test_onlyPrediction.py -indir ../XGB_Input -outdir ../results -model ../Saved_XGBoost -fs CCSandDM3SPE -sc SBCE -sm TopF_Model -inputname All_11E_CCSandDM3SPE.pkl -chrname 8_PatientLevel_char_WithPossibleMonthsHasNoCodes.xlsx -cutoff 0.6
